我们现在使用的博客框架是[Hexo](https://hexo.io/zh-cn/)，感兴趣的同学可以去了解下。

### 下载源码
从GitHub的仓库中下载博客工具源码到自己机器上

```bash
git clone https://github.com/THS-FE/ths-fe.github.io.git
```

### 安装依赖
进入项目所在文件夹，执行
```bash
npm install -g hexo-cli
npm install
```

### 运行Hexo
```bash
hexo s
```
